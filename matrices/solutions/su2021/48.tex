If $\vec{A}$ is symmetric, then 
\begin{align}
\vec{A}^T =\vec{A}
\label{matrix/48/2.0.1}
\end{align}
If matrix A is skew symmetric, then
\begin{align}
\vec{A}^T = \vec{-A}
\label{matrix/48/2.0.2}
\end{align}
From  \eqref{matrix/48/2.0.1} and \eqref{matrix/48/2.0.2},
\begin{align}
\vec{A} &=\vec{-A}
\\
\implies 2\vec{A} &= 0
\\
\text{or, } \vec{A} &= 0
\end{align}
$\therefore $ $\vec{A}$ is zero matrix.
